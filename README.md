# Sales Dashboard – Power BI

## Overview

This project presents an interactive Sales Dashboard developed in Power BI to analyze sales performance and provide actionable business insights. The dashboard enables users to monitor key performance indicators (KPIs), identify sales trends, and make data-driven decisions through dynamic visualizations.

## Key Features

* Interactive KPI cards for Total Sales, Revenue, Orders, and Customers
* Monthly sales trend analysis using line charts
* Sales distribution by age group and other customer segments
* Dynamic filtering with slicers for easy data exploration
* Clean and user-friendly dashboard design

## Data Cleaning

The dataset was cleaned and transformed before visualization using:

* Converting data types (dates, numbers, and text)
* Creating calculated columns (e.g., Month Name and Age Group)
* Preparing the dataset for accurate analysis

## Tools Used

* Power BI
* Microsoft Excel
* Power Query

## Skills Demonstrated

* Data Transformation
* Data Modeling
* Interactive Dashboard Design
* Business Intelligence and Data Visualization

## Key Insights

* Generated **456K** in total revenue from **1K customers** and **1K orders**.
* **Electronics** is the highest revenue-generating category (**157K**), while **Clothing** has the highest quantity sold (**894 units**).
* Sales peaked in **May, October, and December**, with the lowest revenue recorded in **September**.
* **Female customers (51.06%)** contributed slightly more revenue than **male customers (48.94%)**.
* Customers aged **46–55** generated the highest revenue.
* Interactive filters by **Month**, **Age Group**, and **Product Category** enable detailed sales analysis and support data-driven decision-making.


## Project Outcome

The dashboard provides a clear overview of sales performance, helping stakeholders track business metrics, identify trends, and support informed decision-making through interactive reports.
